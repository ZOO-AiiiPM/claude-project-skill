# Journal — {PROJECT_NAME}

> 最新条目在顶部（**倒序**，session 开头 Claude 读前几条就知道最近状态）。
> 格式：`## YYYY-MM-DD 标题` + 做了 / 坑了 / 学到，可选 + 决策。
> **真实 > 完整**：没实质进展的日子空着比凑数好。"今天 review 代码"这种条目会稀释 journal 价值。
>
> 审视时机：每周回看上周条目的"坑了"字段，找**重复出现的词** —— 那就是该蒸馏成 rule 或 lesson 的模式。
